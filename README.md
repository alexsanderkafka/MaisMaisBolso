# Tecnologias utilizadas

## Front end
- Dart 3.4.3
- Flutter 3.22.2

## Banco de dados
- Usando o armazenamento local com a  bilbioteca sqflite

# Como executar o projeto
Pré-requisitos: 
- Dart 3.4.3
- Flutter 3.22.2
- Emulador configurado

```bash
#baixar as bibliotecas
flutter pub get

#rodar o projeto
flutter run
```
